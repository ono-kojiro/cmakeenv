.. _label-setup:

###################################
セットアップ
###################################


************************
ダウンロード
************************

以下のようにgitリポジトリからソースコードを取得する。::

   $ git clone https://github.com/ono-kojiro/cmakeenv.git ~/.cmakeenv
   $ cd ~/.cmakeenv

************************
環境変数設定
************************

取得したソースコードのフォルダに含まれている cmakeenv.bashrcを読み込み、環境変数を設定する。

::

  $ source ~/.cmakeenv/cmakeenv.bashrc
 
************************
バージョン確認
************************

shellを再起動し、pyenvのバージョンを確認する。

::

  $ exec $SHELL

  $ cmakeenv version
  system

************************
その他のコマンド
************************

ヘルプメッセージを表示する。

::

   $ cmakeenv help
